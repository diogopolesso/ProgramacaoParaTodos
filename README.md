# Programação para Todos- Site Educativo para Iniciantes

## Descrição do Projeto

Este é um site educacional completo e amigável para pessoas que nunca tiveram contato com programação. O projeto foi desenvolvido com o objetivo de ensinar as principais linguagens de programação de forma simples, objetiva e progressiva.

## Objetivo

Democratizar o acesso ao ensino de programação, oferecendo conteúdo gratuito, fácil de entender para iniciantes na área de tecnologia.

## Estrutura do Site

### Páginas Principais

1. **index.html** - Página inicial com apresentação e escolha de linguagens
2. **html.html** - Tutorial completo de HTML para iniciantes
3. **css.html** - Guia de CSS com exemplos práticos
4. **javascript.html** - Introdução à programação com JavaScript
5. **python.html** - Python para iniciantes com exercícios práticos

### Arquivos de Estilo

- **css/style.css** - Estilos principais responsivos e modernos

## Funcionalidades Implementadas

### Página Principal (index.html)
- Hero section com call-to-action
- Seção sobre os benefícios de aprender programação
- Cards de linguagens com descrições e links
- Navegação responsiva com efeitos de scroll
- Design moderno e acessível

### Página HTML (html.html)
- Introdução ao conceito de HTML
- Explicação de tags e estrutura básica
- Exemplos de elementos HTML (títulos, parágrafos, links, imagens)
- Exercício prático para criar primeira página
- Dicas importantes para iniciantes

### Página CSS (css.html)
- Conceitos de estilização e cores
- Explicação de seletores e propriedades
- Exemplos de layout e responsividade
- Demonstração de pseudo-classes
- Exercício para estilizar a página HTML

### Página JavaScript (javascript.html)
- Introdução à lógica de programação
- Variáveis, condicionais e loops
- Funções e arrays
- Manipulação do DOM
- Demonstração interativa no próprio site
- Exercício com eventos e cliques

### Página Python (python.html)
- Sintaxe simples e legível
- Variáveis e tipos de dados
- Estruturas de controle (if, for, while)
- Funções e coleções (listas, dicionários)
- Exemplo prático de programa de lista de tarefas
- Aplicações reais de Python

## Destaques do Projeto

2. **Progressão Lógica** - HTML → CSS → JavaScript → Python
